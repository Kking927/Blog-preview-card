### What I learned

During this project, I focused on creating a maintainable design system within CSS using a typography scale and spacing tokens. 

A major part of my learning involved working with **Figma** design files:
- **Design Translation:** I learned how to extract exact values for colors, spacing, and border-radius from the Figma inspector to ensure pixel-perfect accuracy.
- **Typography Presets:** I practiced identifying font styles and weights in Figma and mapping them to reusable CSS utility classes.

I also implemented advanced hover states where multiple elements react differently to provide clear visual feedback.

### Continued development

In future projects, I want to explore:

- **BEM Methodology & Scalability:** I plan to refine my use of BEM to ensure my CSS remains modular and easy to maintain as projects grow in complexity. I want to focus on creating "self-contained" components that can be reused across different pages without style conflicts.
- **Advanced Accessibility:** While I used semantic HTML here, I want to dive deeper into ARIA attributes and screen reader testing to ensure my interactive components are fully inclusive for all users.
- **Micro-animations:** I plan to continue exploring transition timing and cubic-bezier curves to make UI interactions feel even more natural and polished.
